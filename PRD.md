# 行程问题可视化工具 - 产品需求文档（PRD）

**版本：** v1.0 MVP
**日期：** 2026-04-23
**产品类型：** 教育辅助工具
**目标用户：** 数学老师、学生

---

## 1. 产品概述

### 1.1 产品定位
帮助数学老师通过动态动画演示行程问题（相遇、追击、往返等），让学生直观理解抽象的运动过程。

### 1.2 核心价值
- **可视化**：将抽象的数学问题转化为直观动画
- **灵活性**：老师可自定义参数，适配不同题目
- **便捷性**：纯HTML文件，双击即用，无需安装

### 1.3 设计风格
**参考：Notion设计系统**
- 温暖米色/奶油色背景
- Serif标题字体（编辑风格）
- 柔和的卡片表面
- 棕色/米色调色板
- 极简但温暖的视觉语言

---

## 2. 功能范围（MVP - 完整版）

### 2.1 核心场景 ✅

#### 场景1：环形跑道追击
- **描述**：多人同向/反向在环形跑道上运动
- **参数**：
  - 跑道长度（可配置）
  - 2-4个物体
  - 每个物体的速度、颜色、起始位置
- **动画**：椭圆跑道，实时位置，圈数统计
- **示例**：甲6km/h，乙8km/h，400米跑道，多久追上？

#### 场景2：直线相遇问题
- **描述**：两地出发，相向而行
- **参数**：
  - 两地距离（可配置）
  - 2-4个物体
  - 每个物体的速度、方向、起始位置
- **动画**：直线跑道，相遇标记
- **示例**：A、B相距225km，甲60km/h，乙72km/h，多久相遇？

#### 场景3：直线往返运动 🆕
- **描述**：物体在两地间往返运动
- **参数**：
  - 两地距离
  - 2-4个物体
  - 自动掉头逻辑
- **动画**：到达端点自动转向，方向箭头切换
- **示例**：甲从A出发到B后返回，乙从B出发到A后返回，第二次相遇？

#### 场景4：复杂走走停停 🆕
- **描述**：物体有规律的"行驶-停顿"循环
- **参数**：
  - 每个物体独立的停顿规则
  - 行驶时长（分钟）
  - 停顿时长（分钟）
- **动画**：停顿时图标变灰，倒计时显示
- **示例**：甲跑45分钟停5分钟，乙跑25分钟停5分钟，多久相遇？

```
┌─────────────────────────────────────────────────────────┐
│ 🚗 物体1 (甲)                          [🗑️ 删除物体]  │
├─────────────────────────────────────────────────────────┤

│ 速度: [6] km/h                         起始位置: [0%]  │

│                                                      │
│ ☑ 启用停顿规则                                        │
│   每行驶 [45] 分钟，停 [5] 分钟                        │
└─────────────────────────────────────────────────────────┘
```

#### 3.2.3 配置项说明

**物体参数：**
- 名称：文本输入（如：甲、乙、甲车、乙车）
- 速度：数字输入（km/h）
- 颜色：颜色选择器（预设10种颜色）
- 起始位置：滑块 0-100%（或数字输入）
- 方向：
  - 环形：顺时针 / 逆时针
  - 直线：从A向B / 从B向A

**停顿规则（简单模式）：**
- 复选框：启用/禁用
- 行驶时长：数字输入（分钟）
- 停顿时长：数字输入（分钟）

### 6.3 核心算法

#### 环形跑道位置计算
```javascript
// 速度差追上时间
function calculateCatchUpTime(speed1, speed2, trackLength) {
  const speedDiff = Math.abs(speed2 - speed1);
  return trackLength / speedDiff;
}
```

#### 直线相遇时间计算
```javascript
// 相向而行相遇时间
function calculateMeetingTime(speed1, speed2, distance) {
  const relativeSpeed = speed1 + speed2;
  return distance / relativeSpeed;
}
```

#### 走走停停状态计算
```javascript
// 计算当前是行驶还是停顿
function getStopAndGoStatus(elapsedTime, runDuration, stopDuration) {
  const cycleTime = runDuration + stopDuration;
  const timeInCycle = elapsedTime % cycleTime;
  return timeInCycle < runDuration ? 'running' : 'stopped';
}
```

## 7. 预设题目清单

### 7.1 基础题（4题）

#### 题目1：环形追击-同向
```
场景：环形跑道
跑道长度：400米
物体1：甲，6 km/h，从起点出发，顺时针
物体2：乙，8 km/h，从起点出发，顺时针
问题：乙多久能追上甲？
答案：12分钟
```

#### 题目2：环形追击-反向
```
场景：环形跑道
跑道长度：400米
物体1：甲，6 km/h，从起点出发，顺时针
物体2：乙，8 km/h，从起点出发，逆时针
问题：多久后相遇？
答案：约1.64分钟
```

#### 题目3：直线相遇-基础
```
场景：直线相遇
两地距离：225公里
物体1：甲车，60 km/h，从A向B
物体2：乙车，72 km/h，从B向A
问题：多久后相遇？
答案：约102分钟
```

#### 题目4：直线往返-简单
```
场景：直线往返
两地距离：300公里
物体1：甲车，60 km/h，从A向B，到B后返回
物体2：乙车，90 km/h，从B向A，到A后返回
问题：第二次相遇在什么时候？
答案：约400分钟（具体计算）
```

### 7.2 进阶题（3题）
